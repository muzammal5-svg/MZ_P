## 🚀 Quick Setup (5 Minute)

### **Step 1: Resend Account Banayein** (2 minute)

1. **Resend Website Kholen:**
   - Yahan jayen: https://resend.com
   - **"Sign Up"** par click karen (upar right side)

2. **Sign Up Karen:**
   - Apna email dalen: `muzammalzaheer571@gmail.com`
   - Password banayein
   - "Sign Up" par click karen
   - Email check karen aur verify karen

3. **Login Karen:**
   - Wapas jayen: https://resend.com
   - Login karen

---

### **Step 2: API Key Len** (1 minute)

1. **Login Ke Baad:**
   - Resend dashboard dikhega
   - Left side mein **"API Keys"** par click karen

2. **API Key Banayein:**
   - **"Create API Key"** button par click karen
   - Name: `Portfolio Contact Form`
   - Permissions: Default rakhen (Full Access)
   - **"Create"** par click karen

3. **API Key Copy Karen:**
   - Aapko key dikhegi: `re_xxxxxxxxxxxxxxxxxxxxx`
   - **IS KEY KO COPY KAR LEN** (dobara nahi milegi!)
   - Safe jagah rakhen

---

### **Step 3: Vercel Mein Add Karen** (2 minute)

1. **Vercel Kholen:**
   - Yahan jayen: https://vercel.com
   - Login karen
   - Apna **"MZ_P"** project kholen

2. **Settings Mein Jayen:**
   - **"Settings"** tab par click karen (upar)
   - **"Environment Variables"** par click karen (left side)

3. **Variable Add Karen:**
   - **"Add New"** button par click karen
   - Yeh bharen:
     - **Name**: `RESEND_API_KEY`
     - **Value**: Apni Resend API key paste karen (re_xxxxx...)
     - **Environment**: Teeno checkboxes select karen (Production, Preview, Development)
   - **"Save"** par click karen

4. **Redeploy Karen:**
   - **"Deployments"** tab par jayen
   - Latest deployment ke 3 dots (...) par click karen
   - **"Redeploy"** par click karen
   - 2-3 minute wait karen

## 🔧 Problems Aur Solutions

### **Problem 1: Email Nahi Aayi**

**Check Karen:**
1. Resend API key Vercel mein sahi hai
2. API key add karne ke baad redeploy kiya
3. Spam folder check kiya
4. Resend dashboard mein delivery status dekhen

**Solution:**
- Resend dashboard kholen
- "Emails" par click karen
- Dekhen email send hui ya nahi
- Delivery status check karen

### **Problem 2: API Key Error**

**Error**: "Invalid API key"

**Solution:**
1. Resend dashboard mein jayen
2. Nayi API key banayein
3. Vercel mein update karen
4. Redeploy karen

### **Problem 3: Form Submit Hota Hai Par Email Nahi**

**Vercel Logs Check Karen:**
1. Vercel dashboard → Apna project
2. "Logs" par click karen
3. Error messages dhundhen
4. Check karen API key use ho rahi hai ya nahi
